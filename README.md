- Chrome & Firefox fingerprint emulation (canvas, WebGL, fonts, audio, hardware, etc.)

- Temp-mail / custom email / IMAP verification modes
- Github auto-sync for latest solvers & evasion configs
- Igyx animated dark theme
- Enesygurs modern UI components

## 🚀 Quick Start

1. Open the program
2. Toggle what you need:

   - **Hotmail** — main creation mode
   - **Email** — custom domain or default @outlook.com/@hotmail.com
   - **Api** — temp-mail service or recovery email
   - **Gmail** — fallback mode (rarely used)
   - **Proxy** — load list or single proxy
   - **Delay** — realistic timing (4–12 seconds recommended)
   - **Bots** — how many accounts to create at once

3. Press **Start**

## 🛡️ High Success Tips

- Residential / mobile proxies → datacenter proxies die fast
- Rotate fingerprints every 5–15 accounts
- Realistic delays + IETF/behavior simulation = much better pass rate
- Don't go crazy with Bots count (20+ → instant flags)
- Sync Github configs regularly
- Change proxy pools every few hours
- Avoid same ASN / IP range for long sessions
